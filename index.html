Major Update

9/18/021 - . Conversion to "/" and other small updates

  • CHANGE

    Mewbot now uses slash commands instead of text commands.

    Type / and then the command such as "/info", for all other Mewbot commands. If you do not see Mewbot's commands when you type /, check and make sure you see the mewbot logo in the pop up menu-click it-and you should start to see commands come up. If you still cannot see mewbot commands after trying that, check that you have the "Use Application Commands" permission in the server, this can be overridden by channel permissions as well. If you do, and still cannot see Mewbot's commands, you may need to reinvite the bot to give it permission to use slash commands. Have the owner of the server run ;slashinvite to get an invite link that will give the bot the permissions it needs for slash commands. You will need to kick and reinvite the bot using that link to ensure permissions are correct. Your data will not be lost. If you need any additional help, join the support server by clicking the link above.
  • FIX
    Fixed an error in the displayed stats for all pokemon at max level with EV's set. Previously, stats would sometimes be 1 point higher than they should have been at max level with EV's set..

MAJOR UPDATE

From 8/15/21 to 9/8/21

  • CHANGE 8/15/21

    Duels has been rewritten

    , and should function much more closely to how it does in the games. In this rewrite, commands have been changed to ;duel, ;duel party, and ;duel npc.
  • FIX 9/6/21
    Slowking-galar, and Slowbro-galar are now fixed! They can spawn, be redeemed, and bred properly. -Sorry it took so long!
  • CHANGE 8/26/21

    Update to ;party command.

    You can now save your current party with ;party register and then load it at any time with ;party load . To update a saved party, use ;party register making sure to use the name you used before, and it will update that party with your current one. You can also use ;party list to show all of your saved party's.
